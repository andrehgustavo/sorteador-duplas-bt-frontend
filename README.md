# Sorteador de Duplas - Frontend

## Descrição

O **Sorteador de Duplas - Frontend** é a interface web do sistema auxiliar desenvolvido para o **II Maduro Open de Beach Tênis**. Esta aplicação React permite a visualização das duplas sorteadas e a administração dos sorteios.

### Funcionalidades

- **Administrador:**
  - Realizar o sorteio das duplas.
  - Gerenciar as classificações dos jogadores.
  - Executar o sorteio especial de prêmio.
- **Usuários não logados:**
  - Apenas visualizar as duplas sorteadas.

## Tecnologias Utilizadas

- **React 18.3.1**
- **React Router DOM 7.1.5**
- **Material UI 6.4.3**
- **Axios 1.7.9**

## Configuração do Projeto

### 1. Clonar o Repositório

```sh
 git clone https://github.com/andrehgustavo/sorteador-duplas-bt-frontend.git
 cd sorteador-duplas-bt-frontend
```

### 2. Instalar Dependências

```sh
npm install
```

### 3. Rodar a Aplicação

```sh
npm start
```

A aplicação estará disponível em `http://localhost:3000`.

## Estrutura de Pastas

```
/src
  /components  # Componentes reutilizáveis
  /pages       # Páginas principais
  /services    # Integração com a API (Axios)
  /routes      # Configuração das rotas
```
